REBORN 개인정보처리방침

최종 업데이트: 2026-06-27 · 앱: REBORN (com.rebornai.app) · 문의: rebornai.app@gmail.com

REBORN(이하 "앱")은 사용자의 프라이버시를 최우선으로 설계되었습니다. 본 앱은 기본적으로 어떠한 개인정보도 수집하거나 외부 서버로 전송하지 않습니다. 다만 사용자가 직접 본인의 AI API 키를 입력해 선택적 클라우드 AI 기능을 사용하는 경우에 한해, 입력하신 텍스트와 프로필 정보가 제3자(Google)로 전송됩니다. 자세한 내용은 아래 6항을 참고하세요.

1. 수집하지 않는 정보

앱은 다음을 일절 수집·접근하지 않습니다:

  • 이름·전화번호·이메일 등 신원 정보
  • 얼굴 사진 및 얼굴 인식 데이터
  • 음성 녹음 및 오디오
  • 위치 정보
  • 생체 정보
  • 연락처·캘린더
  • 기기 식별자 및 광고 ID

2. 기기 내 저장 (온디바이스)

닉네임, 자가 진단 결과, 캐릭터 설정, 퀘스트 진행도 등 앱 사용 데이터는 오직 사용자 기기 내부 저장소에만 보관됩니다. 앱은 기본적으로 오프라인으로 동작합니다(선택적 AI 기능을 사용하지 않는 한 외부 전송이 없습니다. 6항 참고).

체형 스타일(사진) 기능: 사용자가 카메라/갤러리로 선택한 사진 1장을 기기 내부에서만 1회 분석(온디바이스 자세 추정)하여 옷차림을 추천하며, 원본 사진을 저장하거나 외부로 전송하지 않습니다. 얼굴 인식·생체 식별을 하지 않으며, 분석 후 사진 데이터는 메모리에서 사라집니다.

3. 제3자 제공 및 공유

기본 동작에서는 제3자에게 제공·판매·공유되는 정보가 없습니다. 다만 사용자가 선택적 클라우드 AI 기능을 켠 경우, 입력하신 텍스트와 프로필 정보(닉네임·나이·직업·지역·관심사·목표 등)가 답변 생성을 위해 사용자가 선택한 AI 제공자(Google LLC · Anthropic · OpenAI 중)로 전송됩니다(6항). 그 외 어떤 정보도 광고·판매 목적으로 공유하지 않습니다.

4. 데이터 삭제 (회원탈퇴)

앱(REBORN, 개발자: REBORN)의 모든 사용자 데이터는 사용자 기기에만 저장되며, 개발자 서버에 보관되는 데이터는 없습니다.

삭제 방법(단계): 앱 실행 → 하단 탭/메뉴의 설정 → 데이터 전체 삭제 / 회원탈퇴 버튼 → 확인. 이 한 번의 동작으로 아래 모든 데이터가 즉시 영구 삭제되며 복구할 수 없습니다.

삭제되는 데이터 유형: 닉네임·프로필(나이·직업·지역·관심사·목표), 진단/자가체크 결과, 캐릭터·능력치, 퀘스트·미션 진행도, 출석·보상 기록, 코치 대화 내역 및 자동 추출 메모.

보관 기간: 위 데이터는 사용자 기기에만 있으므로 별도 서버 보관 기간이 없습니다(보관 0). 앱을 삭제(언인스톨)해도 모든 로컬 데이터가 함께 제거됩니다. 선택적 AI 코치 사용 시 외부 AI 제공자로 전송된 입력 텍스트의 보관·삭제는 사용자가 직접 선택한 제공자(Google·Anthropic·OpenAI)의 정책을 따릅니다.

5. 광고 및 결제

본 앱에는 광고가 없으며, 광고 ID를 사용하지 않습니다. 현재 인앱 결제가 없습니다. 향후 결제가 도입될 경우 Google Play 결제 정책을 준수하고 본 방침을 갱신합니다.

6. AI 기능 고지

AI 코치·말투 분석·레드플래그 점검·사주 풀이 등 기본 기능은 모두 기기 내부의 규칙 기반 로직으로 동작하며 외부로 데이터를 전송하지 않습니다.

선택적 클라우드 AI(BYO 키): 사용자가 직접 본인의 AI API 키(Google Gemini, Anthropic Claude, 또는 OpenAI 중 선택)를 입력한 경우에 한해, AI 챗 사용 시 입력하신 텍스트와 함께 맞춤 답변을 위한 프로필 정보(닉네임·나이·직업·지역·관심사·목표·성격 메모·앱 내 누적 기억)가 선택한 제공자(Google LLC, Anthropic, 또는 OpenAI)로 전송됩니다. 이는 전적으로 사용자의 선택이며, 키를 입력하지 않으면 어떠한 데이터도 외부로 전송되지 않습니다. 이 경우 해당 제공자의 개인정보처리방침 (Google · Anthropic · OpenAI)이 함께 적용됩니다. 키는 사용자 기기에만 저장됩니다.

7. 아동 보호

본 앱은 성인 사용자를 대상으로 하며 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 방침 변경

본 방침은 관련 법령 또는 기능 변경에 따라 갱신될 수 있으며, 변경 시 본 페이지에 게시합니다.

Privacy Policy (English summary)

REBORN collects no personal data by default and has no account system. All data (nickname, diagnosis results, character, quest progress) is stored only on your device. We do not collect identity, face/voice/biometric, location, contacts, or advertising IDs, and there are no ads. Core AI features (coach, speech analysis, red-flag check) run on-device (rule-based) with no external calls. Optional cloud AI (bring-your-own-key): only if you enter your own AI API key (Google Gemini, Anthropic Claude, or OpenAI), your chat text and profile info (nickname, age, job, region, interests, goals, saved memories) are sent to your chosen provider (Google LLC, Anthropic, or OpenAI) to generate replies; that provider's Privacy Policy then also applies. You can permanently delete all local data anytime via Settings > Delete all data. Contact: rebornai.app@gmail.com
